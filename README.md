# OurGLP1 v2

A React + TypeScript + Vite + Capacitor mobile app for GLP-1 tracking, reminders, and subscription-based features.

---

## 🧱 Tech Stack

- React (Vite)
- TypeScript (strict mode)
- Capacitor (iOS / Android)
- RevenueCat (subscriptions)
- Local-first auth + storage
